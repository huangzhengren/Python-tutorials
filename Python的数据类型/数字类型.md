# 数字类型

## 整型(int)

```python
age_of_qiqi = 25
print(age_of_qiqi)
print(type(age_of_qiqi))
```

```
25
<class 'int'>

```

## 浮点型(float)

```python
salary = 520.1314
print(salary)
print(type(salary))
```

```
520.1314
<class 'float'>

```

## 复数类型(complex)

```python
complex_example = (520 + 1314j)
print(complex_example)
print(type(complex_example))
```

```
(520+1314j)
<class 'complex'>

```
