# 数字类型

## int类型(int)

```python
age_of_qiqi = 25
print(age_of_qiqi)
print(type(age_of_qiqi))
```

```
25
<class 'int'>

```

## float类型(float)

```python
salary = 520.1314
print(salary)
print(type(salary))
```

```
520.1314
<class 'float'>

```

## bool类型(bool)

* True

  ```python
  is_girl = True
  print(is_girl)
  print(type(is_girl))
  
  ```

  ```
  True
  <class 'bool'>
  
  ```

* False

  ```python
  is_boy = False
  print(is_boy)
  print(type(is_boy))
  ```

  ```
  False
  <class 'bool'>
  
  ```

## 复数类型(complex)

```python
complex_example = (520 + 1314j)
print(complex_example)
print(type(complex_example))
```

```
(520+1314j)
<class 'complex'>

```
