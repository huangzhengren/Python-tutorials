### Python的输出

* 示例1

```python
print("琪琪")
print("好久不见")
```

```
琪琪
好久不见

```

* 示例2

```python
print("琪琪", end="")
print("好久不见", end="")
```

```
琪琪好久不见
```

* 示例3

```python
print("琪琪", end="，")
print("好久不见", end="。")
```

```
琪琪，好久不见。
```

* 示例4

```python
print("琪琪", end="，\n")
print("好久不见", end="。\n")
```

```
琪琪，
好久不见。

```
